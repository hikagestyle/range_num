## 何ができる？

range関数で、正順・逆順の連番を生成する。(1から5万の連番でも1秒で生成)

※初期値は、1から3000

- 正順）for i in r:
- 逆順）for i in reversed(r):
- コメントアウト[#]で連番方法を決める

生成されたものは、表計算ソフト（ExcelやCalc）に貼り付けることが多い。
